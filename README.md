# Clove Dental Landing Page

A responsive landing page built from a Figma design for a dental clinic website.  
This project is developed using HTML, CSS, and vanilla JavaScript.

---

## Links

Live Demo: https://sanju20024.github.io/clove-dental-landing/  
Repository: https://github.com/sanju20024/clove-dental-landing

---

## Project Overview

Clove Dental Landing Page is a static front-end project that replicates a real-world dental clinic website.  
The project focuses on responsive design, accessibility, and interactive UI components.

This is a front-end demo project.  
Form submissions are mocked using localStorage and no backend is connected.

---

## Features

- Responsive design for mobile, tablet, and desktop
- Sticky header with navigation
- Hero carousel with auto-play and infinite looping
- Booking form with client-side validation
- Services section with modal popups
- Video testimonials with modal playback
- FAQ accordion with search functionality
- Dark and light theme toggle using localStorage
- Smooth animations and transitions

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Browser APIs
  - IntersectionObserver
  - localStorage
  - requestAnimationFrame

## How to Run Locally

### Step 1: Clone the repository

git clone https://github.com/sanju20024/clove-dental-landing.git

cd clove-dental-landing

### Step 2: Start a local server

python -m http.server 5173

### Step 3: Open in browser

http://localhost:5173
